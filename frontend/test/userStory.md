## Test 1: Arrivée d'un premier utilisateur
Un utilisateur arrive sur le site et constate que la barre de recherche ne contient que le mot Accueil.
Il clique sur s'enregistrer
L'utilisateur essaie de soumettre son enregistrement en oubliant un champ. 
On reste sur la page et un message d'erreur apparaît.
Il remplit ensuite tous les champs et clique sur Soumettre.
Il clique sur le bouton cliquer ici pour se logger.
Il essaie de se logger mais se trompe de mot de passe. Un message d'erreur apparait.
Il entre ensuite les bons identifiants et arrive sur la page "Te décrire".
Il remplit les données demandées et clique sur sauvegarder les informations.
Il arrive alors sur la page Tes établissements. 
Il n'y aucun établissement dans la liste. Il clique donc sur "Créer un établissement".
Il remplir les données du formulaire et clique sur Sauver l'établissement.
L'établissement est enregistré et il reste sur la même page.
Il constate que les boutons Accueil, Mon profil, Mes expérimentations sont apparus. 
Il clique sur quitter la page et arrive sur la page d'accueil.

## Un utilisateur extérieur visite le site
L'utilisateur arrive sur la page utilisateur, il clique sur l'onglet base de données.
Il clique sur une expérimentation existante d'un utilisateur qui a accepté de partager son contact. Il voit son contact.
Il clique sur le lien pour revenir à la liste des expérimentations et clique sur l'expérimentation d'un utilisateur qui lui n'a pas accepté d'être contacté, son contact n'apparait pas mais un autre champ est visible.
